# Démarrer l'application

## Prérequis

Installer **Node.js** (v18 ou supérieur) depuis : https://nodejs.org/

## Lancer l'application

Ouvrir un terminal dans ce dossier puis :

```bash
npm install
npm run dev
```

L'application sera accessible sur : http://localhost:5173

## Build de production

```bash
npm run build
npm run preview
```
